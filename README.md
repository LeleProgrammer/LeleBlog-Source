# LeleBlog-Source

本仓库为我的个人博客 [leleprogrammer.github.io](https://leleprogrammer.github.io) 的 Hexo 源代码。

---

## 🚀 快速开始

### 1. 克隆本仓库

```bash
git clone git@github.com:LeleProgrammer/LeleBlog-Source.git
cd LeleBlog-Source
```

### 2. 安装依赖

```bash
npm install
```

### 3. 本地预览

```bash
hexo server
```

浏览器访问 `http://localhost:4000` 即可预览。

## 🛠️ 环境要求

- Node.js
- Git
